### Inotify

```
# set inotify
echo fs.inotify.max_user_watches=524288 | sudo tee \
/etc/sysctl.d/40-max-user-watches.conf && sudo sysctl --system
# check inotify
cat /proc/sys/fs/inotify/max_user_watches
```
