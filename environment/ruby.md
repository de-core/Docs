// и в новой вкладке выполнить

```

```

### gemsets

на каждый тип проекта делаем свой gemset
```
rvm gemset create teddy
```

используем гемсет
```
rvm gemset use teddy
```
или
```
rvm use 2.1.1@teddy --default
```
